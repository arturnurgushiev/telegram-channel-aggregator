# Telegram Channel Aggregator

## Суть проекта

Telegram Channel Aggregator решает проблему фрагментации контента - вместо отслеживания десятков каналов в разных чатах, пользователь получает всю информацию в одном месте через удобного бота.

**Ключевая ценность:**
- **Единая лента** - все подписки в одном интерфейсе
- **Централизованные уведомления** - не нужно мониторить каждый канал отдельно

## Пользоватьский интерфейс
```
\sub @channel - подписка на канал
\unsub @channel - отписка от канала
```

## Технический стек
- **Go**, **PostgreSQL**
- **Telegram интеграция**: Bot API для пользовательского взаимодействия + UserBot для мониторинга каналов (MTProto)
- **Архитектура**: Чистая архитектура с разделением на handler/service/repository

## Ключевые особенности


- Асинхронная обработка сообщений через горутины
- Буферизованные каналы для межпроцессного взаимодействия
